# Strategy One Platform Reference

## Overview

Strategy One is Strategy Software's core **AI+BI Analytics Platform**—an all-in-one platform for enterprise business intelligence and analytics. Formerly known as MicroStrategy (the company rebranded to Strategy in 2025).

"Turn complex data into confident decisions with a unified semantic layer, AI-powered analytics, and full freedom from vendor lock-in—all on a modern, cloud-native platform."

**Key Differentiators:**
- Unified platform (not a collection of disparate tools)
- AI+BI integration (accelerate AI with BI rigor)
- Governed semantic layer foundation
- Cloud-native and on-premises deployment options
- Vendor-agnostic, open architecture

### 7. HyperIntelligence

**Overview:**
Inject enterprise intelligence into any web application with zero code.

**Key Capabilities:**
- **Overlay analytics on top of existing web applications**
- No changes to source applications required
- Context-aware data cards that appear on hover
- Support for any web-based application

**How It Works:**
1. Define keywords (e.g., "customer ID", "product code")
2. Map keywords to analytics content (Hyper Cards)
3. Deploy HyperIntelligence browser extension or Chrome extension
4. Users see analytics overlays in Salesforce, SAP, Workday, Gmail, etc.

**Use Cases:**
- Sales reps see customer metrics in Salesforce
- Support agents see customer history in service tickets
- Finance teams see real-time data in ERP systems
- Executives see KPIs in email and documents
